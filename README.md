### Script to convert images to pdf file

```console
$ python main.py -h                                                          
usage: Script to convert .*(jpe?g|png) files to pdf [-h] [-d DIR]

Search .*(jpe?g|png) files from the directory.
Named files must contain digits for order append to pdf.
Result pdf file: <path to folder>.pdf

optional arguments:
  -h, --help         show this help message and exit
  -d DIR, --dir DIR  directory with images

For example: python main.py -d ./images

```
